Build

  • Install all dependencies by run npm i.
  • Run gulp to build styles, icons, templates and scripts.
  • Run gulp dev to start the development server (ะต.g. http://localost:3000) and automatically rebuild files when they are changing.

Add files

Adding a stylesheet

As separate SCSS or CSS

Add the path to a stylesheet in build/config.js in styles.separate section

styles: {
	separate: [
		'path/to/a.scss',
		'path/to/a.css',
	]
}
As common SCSS

Add the path to a stylesheet in styles/common.scss

As common CSS

Add the path to a stylesheet in build/config.js in styles.common section

styles: {
	common: [
		'path/to/a.css',
	]
}

Adding a script

As separate ESM or UMD

Add the script path in build/config.js in styles.separate.esm or styles.separate.umd section

scripts: {
	separate: {
		esm: [
			'path/to/a.esm.js',
		],
		umd: [
			'path/to/a.umd.js',
		]
	}
}
As common ESM

Import the script in scripts/common.js

import './components/esm.js'
As common UMD
scripts: {
	common: {
		umd: [
			'path/to/a.umd.js',
		]
	}
}

Adding an icon

  • Put the SVG file in folder images/icons
  • Run gulp icons
  • Copy the svg snippet from images/icons.html 
    <svg>
	<use xlink:href="#icon"></use>
</svg>
